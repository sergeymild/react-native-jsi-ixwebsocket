# react-native-jsi-ixwebsocket

React Native JSI IXWebSocket library

## Installation

```sh
npm install react-native-jsi-ixwebsocket
```

## Usage

```js
import { multiply } from "react-native-jsi-ixwebsocket";

// ...

const result = await multiply(3, 7);
```
